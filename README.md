# QA Automation Workshop Project

Przykładowy projekt testów automatycznych używany podczas warsztatów z Git.

## Cel projektu
Projekt symuluje repozytorium testów automatycznych dla prostego modułu logowania i dashboardu.

## Struktura
- `tests/` – scenariusze testowe
- `pages/` – page objecty
- `fixtures/` – dane testowe
- `utils/` – pomocnicze funkcje i konfiguracja
- `test-results/`, `playwright-report/`, `screenshots/` – artefakty uruchomień

## Uruchomienie
1. `npm install`
2. `npx playwright test`
